# Jupyter Notebook Python, Scala, R, Spark, Mesos Stack

build databook images for jupyterhub-k8s, running as singleuser notebook.

### include:

- based jupyter/all-spark-notebook
- nodejs 8.10.0
- bs4、lxml、ipyleaflet
- py4j、pyspark
- mlflow、airflow、tushare
- jupyter labextension install jupyter-leaflet
